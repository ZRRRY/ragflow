# 从书籍到知识图谱：操作指南

> 面向没有计算机背景的文科师生，用大白话解释怎么把书籍资料变成一张可视化的知识网络。

---

## 一、整个流程一览

```
书籍资料（Word/PDF/文本）
        ↓
转换成 Markdown 格式（纯文字版本）
        ↓
【第一步：智能切块】→ 变成一张张"知识卡片"
        ↓
【第二步：提取实体】→ 告诉系统要关注哪些实体类型
        ↓
【第三步：生成图谱】→ 自动提取实体，构建知识网络
        ↓
可视化知识图谱（可交互、可查询、可分析）
```

---

## 二、第一步：准备书籍资料

### 2.1 什么是 Markdown？

不用学会它，只需要知道：**Markdown 是一种最干净的文字格式**，没有 Word 里复杂的字体、颜色、排版，只剩下纯粹的文字内容和标题层级。

> 就像把一本排版华丽的书，还原成一份只有标题和正文的"纯文字手稿"。

### 2.2 需要做什么？

把要分析的书籍资料发过来，由技术侧成员转换成 Markdown 格式。

**可以提供的资料格式：**
- Word 文档（.doc/.docx）
- PDF 文件
- 网页文章
- 纯文本文件
- 甚至自己整理的笔记

---

## 三、第二步：智能切块（系统自动完成）

### 3.1 为什么要"切块"？

一本书可能有几万字甚至几十万字。计算机无法一次性"消化"这么长的内容。

> 就像让一位学生总结一本书，不会要求他一口气读完就写报告，而是会要求他按章节做读书笔记，最后再把笔记整合起来。

**切块**就是把长文章按照内容的自然边界（如章节、段落、主题）切成一段段长短适中的"知识卡片"。每段卡片包含一个相对完整的小主题。

### 3.2 系统怎么切？

系统会智能识别文章的结构：

| 识别依据 | 举例 |
|---------|------|
| 标题层级 | 第一章、第二节、（三）…… |
| 自然分段 | 空行分隔、换行处等 |

**举例说明：**

假设原始文本是：

```
## 第三章 宝玉初会黛玉

宝玉初见黛玉，便觉似曾相识……（此处五百字描写）

## 第四章 宝钗进京

薛宝钗随母亲哥哥进京，暂居贾府……（此处八百字描写）
```

系统会切成两张"知识卡片"：

- **卡片 A**：第三章 宝玉初会黛玉（含该章内容）
- **卡片 B**：第四章 宝钗进京（含该章内容）

### 3.3 需要做什么？

**这一步完全由系统自动完成，无需操作。**

如果对切分结果有特殊要求（比如按固定字数切），可以告诉技术侧成员调整参数。

---

## 四、第三步：知识图谱生成

### 4.1 什么是"知识图谱"？

其实就是一张**概念关系网络图**。

**举个例子：**

如果研究的是中国现代文学，知识图谱可能长这样：

```
鲁迅 ──作品──→ 《呐喊》
 │                │
 └──影响──→ 巴金 ──作品──→ 《家》
 │
 └──师生关系──→ 萧红 ──作品──→ 《生死场》
```

在这张图里：
- **圆圈（节点）**：鲁迅、巴金、萧红、《呐喊》、《家》、《生死场》……这些叫做**实体**；
- **连线（边）**："作品"、"影响"、"师生关系"……这些叫做**关系**；
- **整体网络**：把所有实体和关系连起来，就是**知识图谱**。

### 4.2 "实体类型"是什么？怎么写？

**实体类型 = 希望图中出现哪些"类别"的圆圈**

下面是不同学科的示例，供参考：

#### 示例一：中国现当代文学研究

| 实体类型 | 具体例子 |
|---------|---------|
| 作家 | 鲁迅、沈从文、张爱玲 |
| 作品 | 《边城》、《倾城之恋》 |
| 文学流派 | 新月派、山药蛋派、伤痕文学 |
| 文学刊物 | 《新青年》、《小说月报》 |
| 文学概念 | 启蒙、乡土、都市书写 |
| 地理空间 | 北京、上海、湘西 |
| 历史时期 | 五四时期、抗战时期、新时期 |
| 机构/组织 | 中国文联、左联 |

#### 示例二：历史学研究（以近代史为例）

| 实体类型 | 具体例子 |
|---------|---------|
| 人物 | 李鸿章、孙中山、袁世凯 |
| 事件 | 辛亥革命、洋务运动、戊戌变法 |
| 地点 | 南京、武昌、紫禁城 |
| 组织/团体 | 同盟会、北洋军阀、保皇会 |
| 条约/文献 | 《南京条约》、《临时约法》 |
| 概念/思想 | 君主立宪、三民主义、中体西用 |
| 时间阶段 | 晚清、民国初年、北伐时期 |

#### 示例三：哲学研究

| 实体类型 | 具体例子 |
|---------|---------|
| 哲学家 | 孔子、康德、海德格尔 |
| 哲学概念 | 仁、绝对精神、存在与时间 |
| 哲学流派 | 儒家、存在主义、分析哲学 |
| 著作 | 《论语》、《纯粹理性批判》 |
| 方法/工具 | 现象学还原、辩证法、语言分析 |
| 历史时期 | 轴心时代、启蒙运动、现代性 |

**小提示：**
- 实体类型一般在 **5～12 个** 之间比较合适，太少会遗漏重要信息，太多会让图谱过于杂乱；

---

## 五、需要配合的事项

这个环节需要文科合作成员提供**实体类型清单**。因为这是学术判断，只有熟悉书籍内容的人才能准确回答：这本书里有哪些值得提取的核心概念？

| 事项 | 内容 |
|-----|------|
| 实体类型清单 | 参照上面的示例，列出所提供书籍中需要提取的实体类型 |

> 需要仔细思考：这本书里有哪些值得提取的核心概念？这不是技术问题，而是**学术判断**——这正是领域专家不可替代的价值所在。
